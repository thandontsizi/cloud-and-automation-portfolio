# Linux System Administration Lab:

## Overview:
This lab demonstrates practical Linux system administration skills in a simulated server environment.

It focuses on:
- User and group environment.
- File ownerships and permissions.
- Service management,
- Log analysis.
- System diagnostics.

These exercises reflect real operational tasks performed by a systems administrator.
------------------------------------------------------------------------------------

## Projects in This Lab:
### 1. system-setup-project:
Covers:
- Initial system configuration.
- User and group creation.
- File permission management,
- Directory structuring.
------------------------------

### 2. system-audit-project;
Covers:
- Inspecting system state.
- Reviewing disk usage and memory.
- Identifying misconfigurations.
- Auditing system information.
----------------------------------

### 3. process-and-services-project:
Covers:
- Managing system services (systemctl).
- Monitoring running processes,
- Identifying failed services.
- Inspecting logs for troubleshooting.
-------------------------------------

## Skills Demonstrated:
- Confident CLI usage.
- Controlled user provisioning.
- Service lifecycle management.
- Log-based troubleshooting.
- Structured system diagnostics.
--------------------------------

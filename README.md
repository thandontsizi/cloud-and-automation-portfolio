# Infrastructure Engineering Projects:
This repository contains structured production-style labs demonstrating practical infrastructure engineering skills.

Each lab focuses on real operational tasks performed on Linux-based systems.
----------------------------------------------------------------------------

## Project Order:

### 01 - Bash Scripting Lab:
Production-style Bash automation scripts:
- System health reporting.
- Safe user provisioning.
-------------------------

## Philosophy:
These labs are designed to:
- Simulate real server environments.
- Emphasise defensive scripting.
- Prevent silent failures.
- Demonstrate operational awareness.

Learning notes and command drills are kept in the companion repository: 
'infrastructure-engineering-learning'.

This repository strictly focuses on applied implementation.
